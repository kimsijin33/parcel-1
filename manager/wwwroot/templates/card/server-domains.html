The origin is the domain name that locally-posted mail appears to come from, this is most likely the FQDN of the host where parcel is installed on.

A domain or FQDN, for example: example.com or myhost.example.com

List all domains that you like to manage emails for. All domains need to have valid MX records pointing to the server where Parcel is installed.

A comma separated list of one or more domains, for example: example.com,example.net.

A nice tool to detect various configuration errors are MXToolBox.
