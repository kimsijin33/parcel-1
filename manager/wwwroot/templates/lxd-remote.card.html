Install LXD on a server running Linux. If you are running Ubuntu, LXD is already installed. If not lxd is available inside the snap store. You can verify that lxd is installed by running the command lxc (lxd client).

To get started you need to configure LXD to listen to the network and add a password. If you intend to use the local LXD installation set core.https_address to 127.0.0.1, if you are using a separate machine set it to [::]. You also need to protect LXD by setting a good password.

sudo lxc config set core.https_address "127.0.0.1" sudo lxc config set core.trust_password "{{ args.example_password }}"

You need to disconnect the lxd snap if you have previously used it with a sudo snap disconnect parcel:lxd. The system will prioritize the lxd snap over this option if both are configured.

DNS/IP and port number to the LXD server, prefix it with https://. The default port is 8443, you can specify a different port with the lxc command if you like.
