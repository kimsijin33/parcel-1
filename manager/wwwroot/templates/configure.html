This snap uses LXD system containers to run the email software and for various tools. You can install LXD on the same system (easiest) or on a remote server (it's important that this snap has access to the same network). If you are running Ubuntu, LXD is already installed. If not lxd is available inside the snap store. You can verify that lxd is installed by running the command lxc (lxd client).

If you intend to use the local LXD installation set core.https_address to 127.0.0.1, if you are using a separate machine set it to [::]. You also need to protect LXD by setting a good password, for example {{ example_password }}.

sudo lxc config set core.https_address "127.0.0.1" sudo lxc config set core.trust_password "{{ example_password }}"

The default port is 8443, you can specify a different port with the lxc command if you like.
