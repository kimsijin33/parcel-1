Provision logs

Ansible is used to create and provision the LXD containers.

This page displays the status and logs from the latest provision operation. This may fail for a various reasons, if you have not configured the snap under configure, do it now. The status page also displays various information that may help you.

The containers are provisioned on snap install, configuration changes and once every hour. This is done to make sure that configuration and package information is correct and always in a working order.
